Yucca Dashboard

Una collezione di widget AngularJS pronti all'uso

Get Started
Scopri i widget
Prova il composer
Guarda su Github

I widget sono il modo più immediato di visualizzare i dati ospitati sulla piattaforma Yucca

Includi i widget nelle pagine della tua webapp. Le configurazioni sono attributi del widget stesso

Ci sono tante tipologie di widget. Grafici, tabelle, mappe … Tutti interattivi e altamente personalizzabili

Con il composer puoi creare la tua dashboard esportabile direttamente online, componendo e provando i widget in tempo reale

Se non trovi il widget giusto, parti da Github per svilupparne uno tuo
