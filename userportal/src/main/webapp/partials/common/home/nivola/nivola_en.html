Smart Data Platform, the CSI solution for Big Data and the Internet of Things

La Smart Data Platform (SDP) è il nuovo strumento open source che il CSI mette a disposizione di pubbliche amministrazioni e imprese per sviluppare applicazioni in campo IoT e Big Data. SDP &egreave; disponibile come Platform-as-a-Service all'interno di Nivola, il cloud CSO per l'Italia digitale.

BIG DATA STORAGE IN LINEA CON LE INDICAZIONI GDPR

Come funziona?

ECOSISTEMI DIGITALI

Grazie a SDP diverse comunità di utenti possono partire dai dati disponibili per sviluppare i propri progetti di busines, interagendo e scambiandosi informazioni

AREA DI LAVORO

Chi utilizza SDP può gestire una propria area di lavoro per operare su tutte le risorse messe a disposizione: smart object, stream, API, dati.

VISIBILITÀ

Gli utenti di aree di lavoro possono definire in autonomia i livelli di visibilità dei propri dati: possono mantenerli privati, condividerli con altri utenti o pubblicarli in open data

Le funzionalità

SDP mette a disposizione una serie di funzionalità utilizzabili in modalità SELF SERVICE su un portale web dedicato
Creare dataset, smart object e stream mediante l'utilizzo di wizard
Definire la struttura di dati e metadati
Popolare i tenant creati mediante stream, API o CSV
Ricercare nello store i dataset/smart object/stream inseriti
Filtrare i dati cercati secondo le diverse faccette (tipo, ambito, sotto-ambito, tag)
Scaricare le informazioni in formato CSV
Utilizzare le API di accesso ai dati (mediante generazione di URL automatica, sottoscrizione e utilizzo di token di accesso) per utilizzare i dati in applicazioni (cruscotti, siti web)
Gestire gruppi di dataset
Utilizzare i widget della piattaforma per la visualizzazione dei dati sui propri siti
